# Findings

## Authorization and Permission Review

This lab focused on reviewing and modifying Linux file and directory permissions to improve system security and user authorization management.

## Security Actions Performed

- Reviewed existing permissions for files and directories
- Removed unnecessary write permissions
- Restricted group access where appropriate
- Modified hidden file permissions
- Restricted directory execution permissions

## Key Takeaways

This activity demonstrated how Linux permissions can be used to control access to sensitive files and directories. Proper authorization management helps reduce security risks and prevents unauthorized access to system resources.

## Conclusion

Understanding and managing Linux file permissions is an essential cybersecurity skill. This lab strengthened my ability to apply access control principles using Linux command-line tools.
