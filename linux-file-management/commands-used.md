# Commands Used

## Create Logs Directory

```bash
mkdir logs
```

Creates a new subdirectory called `logs`.

---

## Remove Temp Directory

```bash
rmdir temp
```

Removes the empty `temp` directory.

---

## Move File

```bash
mv /home/analyst/notes/Q3patches.txt /home/analyst/reports
```

Moves the file to the reports directory.

---

## Edit File with Nano

```bash
nano /home/analyst/notes/tasks.txt
```

Opens the file using the nano text editor.

---

## Display File Contents

```bash
cat /home/analyst/notes/tasks.txt
```

Displays the contents of the file in the terminal.
