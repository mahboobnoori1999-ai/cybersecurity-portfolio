# Linux File Management

## Project Overview

In this lab activity, I practiced essential Linux file management and command-line operations commonly used in cybersecurity and system administration environments.

The objective of the lab was to strengthen foundational Linux skills by creating and removing directories, moving files between locations, editing files with the nano text editor, and displaying file contents through the command line.

This project demonstrates hands-on experience with Linux system navigation and file management tasks that are important for cybersecurity analysts working in Linux-based environments.

---

## Objectives

The main objectives of this lab were:

- Create and manage directories
- Remove unnecessary directories
- Move files between directories
- Edit text files using the nano text editor
- Display and verify file contents
- Practice Linux command-line navigation

---

## Skills Practiced

- Linux command line
- File management
- Directory management
- Bash shell commands
- Nano text editor
- System navigation

## Tools Used

## Conclusion

This lab improved my understanding of Linux file management and command-line operations. These foundational skills are essential for cybersecurity professionals, especially when working with system files, logs, and administrative tasks in Linux environments.
